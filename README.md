#Learning Angular
First code with Angular.
## Get started
### Set up
``$ npm install -g npm@latest``
``$ npm install -g @angular/cli``
#### Start new project
``$ ng new LAngular --style=scss --skip-tests=true``

### Package

#### Bootstrap 
``$npm install bootstrap@3.3.7 --save``
#### rxjs compat
``$npm install rxjs-compat --save ``
